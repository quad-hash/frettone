FretToneとは

FretToneは、ジャズギター初心者向けの無料Webアプリです。コードトーン・スケール・度数をギター指板上でリアルタイムに確認できます。インストール不要で、ブラウザからすぐ使えます。

何に使えるか

ii-V-I進行のコードトーンを指板上で把握したいとき、各コードの構成音（ルート・3度・5度・7度）をビジュアルで覚えたいとき、スケールの指板上の配置を確認したいとき、アドリブ練習の前に音の場所を整理したいとき——そういった場面で役立ちます。

ジャズギター初心者向けの使い方

コードトーン確認：ルート音とコードタイプ（M7・m7・7・m7♭5・dim7）を選ぶと、そのコードの構成音が指板全体に表示されます。Degree表示にすると「R・3・5・♭7」のように度数で確認できるので、音と度数の結びつきを指板上で覚えられます。

ii-V-I練習：たとえばCキーのii-V-Iなら、Dm7（Root: D / Type: m7）→ G7（Root: G / Type: 7）→ CM7（Root: C / Type: M7）と切り替えながら、各コードトーンの位置関係を確認できます。

スケール確認：ModeをScaleに切り替えると、Major・Natural Minor・Dorian・Mixolydian・Altered・Lydian Dominant・Half-Whole Diminished・Whole Toneのスケールが選べます。アドリブで使うスケールの指板上のパターンを視覚的に把握するのに役立ちます。
